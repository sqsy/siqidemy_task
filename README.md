# Siqidemy Task Gem

Add support for a topic that can be marked as "Closed."

## Installation

Add this line to your application's Gemfile:

    gem 'siqidemy_plugin', github: 'sqsy/siqidemy_task', path: 'vendor/gems/siqidemy_task'

And then execute:

    $ bundle
